01.Chisel初步学习

  • Scala介绍

    Scala是一种支持通用编程范式的编程语言

    所谓的编程范式其实指的是软件开发中用于描述软件结构和运行方式的基本概念和方法

    但选择Scala作为硬件开发也有独特优势：

    1. 它具备托管嵌入式DSL的能力（DSL是一种针对特定领域的编程语言，托管嵌入式DSL是指将嵌入式DSL嵌入到另一个宿主语言中使用）

    2. 数据库全，功能强大

    3. 严格的类型系统，方便初期编译纠错

    4. 有强大的表达和传递功能的方式

      ps: Chisel比Chipel、Chijel和Chicel更顺口—— 一个梗，Chisel全程为Constructing Hardware in a Scala Embedded Language，这里就是把Scala替换成了其他语言，如Python，Java，C等，缩写就也跟着变了

  • Scala语言

    由于Scala语言与js、ts等有很大的相似性，故稍微截取记录

    常量与变量

    创建变量的语句用var（variable）关键词作为开始，常量创建用val（value）关键词。

    变量是可变的，常量是不可变的，尽可能使用常量，减少重用变量带来的错误和难以阅读。

    函数

    由def关键词定义

    函数的参数由一个通过逗号分隔的列表指定，包括参数名，参数类型，可选的是参数默认值。

    需要注意的是，返回值的类型需要给定。

    // 一个简单的缩放函数，把输入乘以2，如times2(3)会返回6
// 只有一行的函数可以省略大括号
def times2(x: Int): Int = 2 * x
  
// 一个更复杂的函数
def distance(x: Int, y: Int, returnPositive: Boolean): Int = {
    val xy = x * y
    if (returnPositive) xy.abs else -xy.abs
}

    • 重载

      函数的参数列表和类型决定了函数的签名，让编译器判断应该调用哪个函数

      *// 重载的函数* 
    
def times2(x: Int): Int = 2 * x 
    
def times2(x: String): Int = 2 * x.toInt 
    
times2(5) 
    
times2("7")

    • 递归和嵌套

       打印倒三角形的x阵列
def asciiTriangle(rows: Int) {
    // 字符串的乘法可以将字符串复制多次
    def printRow(columns: Int): Unit = println("X" * columns) 
    if(rows > 0) {
        printRow(rows)
        asciiTriangle(rows - 1) // 这里是递归调用
    }
}
// printRow(1) // 该函数调用不在作用域内，编译不通过
asciiTriangle(6)

    • for语句

      有to 与until之分，例：

      for (i <- 0 to 10 ){ print(i + " ") }
println()
//从0迭代到10
for (i <- 0 until 10 ){ print (i + " ") }
println()
//从0迭代到9，不包括10

      by能指定增幅，例：

      for (i <- 0 to 10 by 2 ){ print(i + " ") }
println()
//输出0 2 4 6 8 10
      ps：可以使用for作为迭代器，和Java以及Python里面是一样的

  • 阅读Scala代码

    • 包和导入

      package mytools 
class Tool1 { ... }

      当需要引用定义了以上代码的一个文件时，应该这么写：

      import chisel3._ 
import chisel3.iotesters.{ChiselFlatSpec, Driver, PeekPokeTester}

      第一句会把chisel3里面所有的类和方法导入，_表示通配符。

      第二句从chisel3.iotesters中导入了指定的类。

    • Scala是一种面向对象的语言

      根据官方文档，下面是一个类的例子：

      // WrapCounter计数到根据位大小确定的最大值
class WrapCounter(counterBits: Int) {
    
  val max: Long = (1 << counterBits) - 1
  //(1 << counterBits) - 1计算counterBits位可以存放的最大值
  var counter = 0L
  //初始化0L，L表示0是一个Long类型的值，因此counter被推断为Long类型
  def inc(): Long = {  //类方法inc定义为不接受任何参数并返回Long值的方法
    counter = counter + 1
    if (counter > max) {
        counter = 0
    }
    counter//很重要，函数inc返回的是counter的值
  }
  println(s"counter created with max value $max")
  //双引号前s代表了这个字符串是插值字符串$max的值会被max取代
}

      类使用操作如下：

      val x = new WrapCounter(2)
x.inc() // counter自增
    
// 实例x的成员变量对外是可见的，除非被声明为private
if(x.counter == x.max) {              
    println("counter is about to wrap")
}
    
x inc() // Scala允许不使用点，这有助于让嵌入式DSL看起来更自然
